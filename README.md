Автоматизированные тесты для входа в демо-сайт Интернет-банк БСПБ

Этот проект содержит автотесты для проверки входа в [демо-версию интернет-банка БСПБ](https://idemo.bspb.ru).

Используются:
- Python + Playwright
- Pytest
- Page Object Model
- Git + GitHub

---

Требования

Убедитесь, что у вас установлено:

- [Python 3.9+](https://www.python.org/downloads/)
- [Git](https://git-scm.com/)

Что делают тесты?
test_valid_login
Проверяет успешный вход с логином  и паролем 
test_invalid_login
Проверяет, что вход неудачен при неверных данных

Если возникли вопросы — создайте Issue в репозитории или обратитесь к автору.
